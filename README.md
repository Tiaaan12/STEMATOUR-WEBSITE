# STEMATOUR Website

Static Mauban tour website with organized pages, shared styles, and shared JavaScript functionality.

## Structure

- `index.html` - main home page
- `pages/` - inner pages for beach, restaurant, parks, tourist guides, more, and AMCI
- `assets/css/styles.css` - active site styles
- `assets/css/legacy-stylee.css` - original CSS kept for reference
- `assets/js/main.js` - navigation, chat, booking, contact, notifications, and gallery behavior
- `assets/images/` - image assets
- `assets/media/` - audio/video assets
- `archive/` - older demo files that are no longer part of the main site

The old root files `website.html`, `Beach.html`, and `restaurant.html` redirect to the new organized pages.
